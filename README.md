# EV Energy Consumption Prediction

## Overview
A machine learning-based web application that predicts electric vehicle energy consumption based on driving and vehicle parameters.

## Problem Statement
Electric vehicle users often face uncertainty regarding battery usage and travel range. This project helps estimate energy consumption using machine learning models.

## Features
- Predict EV energy consumption
- Interactive Streamlit dashboard
- Real-time user input
- Machine learning based predictions
- User-friendly interface

## Tech Stack
- Python
- Scikit-Learn
- Pandas
- NumPy
- Streamlit
- Matplotlib

## Project Architecture
Dataset → Data Preprocessing → Model Training → Prediction → Streamlit Dashboard
